## MongoDB基本概念
#### 1、mongodb中与关系型数据库的概念对应：
1. table-collection：表/集合
2. row-document: 数据记录行/文档
3. column-field: 列/数据域
4. index-index: mongodb里面也有索引

#### mongodb中的语法
```shell
> use databaseName  # 创建
> db
> show dbs
# 插入第一条数据之前数据库不会被创建
> db.databaseName.insert("key1", "value1")
```
